# TopPhoneParts.lk

TopPhoneParts.lk is a custom-built web platform for a Sri Lankan mobile phone parts business, supporting both **wholesale and retail** customers.

The platform allows customers to browse mobile phone parts, request items, and place orders directly via WhatsApp, while providing an internal admin system for product and dealer management.
